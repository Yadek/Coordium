# Coordium

Быстрый конвертер географических координат из формата **DMS** (градусы-минуты-секунды)
в формат **DMM** (градусы и десятичные минуты).

```
широта  55°45'21.0"С         →
долгота 37°37'02.0"В         →   N55 45.350 E37 37.033
```

## Возможности

- Полушарие распознаётся и по кириллице (`С Ю В З`), и по латинице (`N S E W`).
- Понимает разделитель дробной части `.` и `,` (`9.5` и `9,5`).
- Игнорирует лишние слова и любые символы `° ′ ″ ' "` — берёт три числа и букву полушария.
- Округление до 3 знаков с корректным переносом в градусы при значении минут ≥ 60.
- Один исполняемый файл, без зависимостей и установки.

Готовые сборки — на вкладке [**Releases**](../../releases):

| Windows (64-bit) | `Coordium-windows-amd64.exe` |
| macOS (Apple Silicon, M1/M2/M3) | `Coordium-macos-applesilicon.zip` |
| macOS (Intel) | `Coordium-macos-intel.zip` |

> На macOS распакуйте `.zip` — внутри лежит исполняемый файл `Coordium`.
> Бинарник не подписан, поэтому при первом запуске: правый клик → «Открыть»,
> либо в терминале `xattr -d com.apple.quarantine ./Coordium` и затем `./Coordium`.

## Как пользоваться

1. Запустите программу.
2. Вставьте координаты (широту и долготу — можно прямо со словами «широта/долгота»).
3. Нажмите Enter на пустой строке — получите готовую строку `N.. .. E.. ..`.
4. Можно сразу вставлять следующую пару. Выход — два пустых Enter подряд или Ctrl+C.

Пример:

```
> широта 55°45'21.0"С
> долгота 37°37'02.0"В
>
  N55 45.350 E37 37.033
```
