# Coordium

Быстрый конвертер географических координат из формата **DMS** (градусы-минуты-секунды)
в формат **DMM** (градусы и десятичные минуты).

```
широта  55°45'21.0"С         →
долгота 37°37'02.0"В         →   N55 45.350 E37 37.033
```

## Возможности

- Ввод одной строкой (`48°51'30.81"N 2°21'32.10"E`) или широта и долгота по отдельности —
  формат определяется автоматически, порядок широты и долготы не важен.
- Сразу выдаёт ссылки на **Google Maps** и **Яндекс.Карты** (кликабельны в современных терминалах).
- Результат сразу копируется в буфер обмена.
- Также показывает десятичные градусы (DD).
- Аккуратная подсветка вывода; цвета сами отключаются при выводе в файл и уважают `NO_COLOR`.
- Полушарие распознаётся и по кириллице (`С Ю В З`), и по латинице (`N S E W`).
- Понимает разделитель дробной части `.` и `,` (`9.5` и `9,5`).
- Игнорирует лишние слова и любые символы `° ′ ″ ' "` — берёт три числа и букву полушария.
- Округление до 3 знаков с корректным переносом в градусы при значении минут ≥ 60.
- Один исполняемый файл, без зависимостей и установки.

## Скачать

Готовые сборки — на вкладке [**Releases**](../../releases):

| Платформа | Файл |
|-----------|------|
| Windows, обычный ПК (Intel/AMD) | `Coordium-windows-x64.exe` |
| Windows на ARM | `Coordium-windows-arm64.exe` |
| macOS (Apple Silicon, M1/M2/M3) | `Coordium-macos-applesilicon.zip` |
| macOS (Intel) | `Coordium-macos-intel.zip` |

> На macOS распакуйте `.zip` — внутри лежит исполняемый файл `Coordium`.
> Бинарник не подписан, поэтому при первом запуске: правый клик → «Открыть»,
> либо в терминале `xattr -d com.apple.quarantine ./Coordium` и затем `./Coordium`.

## Как пользоваться

1. Запустите программу.
2. Вставьте координаты — одной строкой или по очереди (широту и долготу).
3. Получите результат, десятичные градусы и ссылки на карты.
   В современных терминалах ссылки кликабельны (Ctrl/Cmd + клик).
4. Выход — два пустых Enter подряд или Ctrl+C.

Пример (одной строкой):

```
> 48°51'30.81"N 2°21'32.10"E

  N48 51.514 E2 21.535
  DD:     48.858558, 2.358917
  Google: https://www.google.com/maps/search/?api=1&query=48.858558,2.358917
  Яндекс: https://yandex.ru/maps/?ll=2.358917,48.858558&z=17&pt=2.358917,48.858558,pm2rdm
```
